# CS484

2 finalized text files (train_features.txt, test_features.txt). Channels 1 and 2 voltages are added together to make a single signal per sample (row). 

Features include:

    1-3: Top 3 voltage values
    4-6: Top 3 peak frequencies
    7: RMS of signal
    8: Class of sample:
    
        1: cyl
        2: hook
        3: lat
        4: palm
        5: sphere
        6: tip
